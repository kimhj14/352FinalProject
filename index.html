Overview

We want the system to be able to identify which instrument (violin, trumpet, or vibraphone) is playing a melody. Our system will try to determine whether a melody is being played by a violin, a trumpet, or a vibraphone by using Mel-frequency cepstral coefficients (MFCCs). Since we have already done this in previous homeworks, we think this will be hopefully straightforward.

We think there are a couple of approaches to doing classification. It seems like the most tried and true method is to do a simple difference between cepstrograms. We may have to do Dynamic Time Warping to correct for length, but be might be able to avoid this by chopping the melody into the same length as our cepstrograms.


Motivation

It could be a useful system because successfully developing a musical instrument identification system (MIIS) will help us learn how human brains classify and identify sounds and how we could then use that knowledge to make a computer system do the same. We think it’s cool because it could make it easy to create a database of songs by instrument type.

Our Approach

Our system will try to determine whether a melody is being played by a violin, a trumpet, or a vibraphone by using Mel-frequency cepstral coefficients (MFCCs).

Data Set

We could use the samples of recordings of violin, Bb trumpet, and vibraphone playing different notes to see whether the system can identify which instrument is being played. These recordings are downloaded from University of Iowa Electronic Music Studios. They are aiff-c audio types in various sizes. They are already labeled. We will also use samples of monophonic melodies played by the three different instruments to test the system.
